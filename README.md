### About

I'm a founding engineer at [Youe](https://youe.co), where I build LLM and agent backends in Python and the React 19 interfaces in front of them. My day-to-day is production AI infrastructure — streaming pipelines, agent orchestration, observability, and the distributed-systems plumbing that keeps it all reliable. I also maintain open-source tooling for the agent ecosystem.

- 🔭 Leading agent and inference platform work at Youe
- 🌱 Deep in agent observability, OpenTelemetry, and real-time voice systems
- 💬 Happy to talk about agent observability, distributed systems, and AWS / Azure platform work
- 📫 Reach me at **me@sayanbiswas.in**

### What I'm building

- **[AxonPush](https://axonpush.xyz)** — open-source observability for agent and inference runs. Ships [Python](https://github.com/AxonPush/python-sdk) and [TypeScript](https://github.com/AxonPush/ts-sdk) SDKs plus an OpenTelemetry exporter, with first-class integrations for LangChain, OpenAI Agents, Anthropic, and CrewAI.
- **Athena @ Youe** — a streaming voice agent for salary-negotiation practice, built on SSE and LiveKit.
- **Open source libraries** — [MusixScrape](https://github.com/Dank-del/MusixScrape) (Go lyrics scraper), [anilistWrapGO](https://github.com/Dank-del/anilistWrapGO) (Go AniList API wrapper), and [eru](https://github.com/Dank-del/eru) (multi-threaded C++ downloader).
